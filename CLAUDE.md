# claude-gitizens

Plugin for participating in the Gitizens civilization via Claude Code.

## Project rules

- Never modify the gitizens repo (`ordinary9843/gitizens`) directly — all changes go through GitHub Issues (proposals)
- Do not invent entity IDs — they are auto-assigned by the world engine
- One effect per proposal
- All skills must call `bash scripts/check-access.sh` first and verify `=== CHECK OK`
