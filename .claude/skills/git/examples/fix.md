# fix — examples

Use when correcting a bug or broken behavior in a skill or hook.

## Good

```
fix: fall back to python3 base64 decode on Windows
fix: handle missing active_event key in session-start
fix: correct voting period from 3 days to 24 hours in README
fix: prevent check-access from failing on Windows paths
```

## Bad

```
fix: fixed the bug              ← past tense, vague
fix: Bug fix                    ← not lowercase, not imperative
fix: various fixes              ← too vague
```
