# feat — examples

Use when adding a new skill, command, or visible capability to the plugin.

## Good

```
feat: add gitizens:laws skill for listing enacted laws
feat: show active event in status skill output
feat: add era progression table to propose skill
feat: display treasury and population in session-start
```

## Bad

```
feat: added new stuff          ← too vague
feat: Skill                    ← not imperative, not lowercase
feat: implement the entire new status skill that shows world state and active events  ← too long
new skill                      ← missing type prefix
```
