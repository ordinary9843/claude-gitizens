# chore — examples

Use for maintenance tasks, config cleanup, renaming — no behavior change.

## Good

```
chore: remove unused debug echo from check-access
chore: rename skills/world to skills/status
chore: add .gitignore for macOS metadata files
```

## Bad

```
chore: update stuff            ← too vague
chore: changes                 ← not imperative
cleanup                        ← missing type prefix
```
